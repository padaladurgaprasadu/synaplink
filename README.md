# 🧠 SynapLink – Brain to Code Interface

SynapLink is an experimental Brain-Computer Interface (BCI) project that aims to convert human brain signals into text or programming code using machine learning and AI.

The system captures neural signals through scalp sensors, processes them using signal processing and ML models, and generates meaningful output such as text or source code.

---

## 🚀 Features
- EEG signal acquisition using scalp electrodes  
- Signal amplification and noise filtering  
- Machine learning–based pattern recognition  
- Brain-to-text and Brain-to-code generation  
- Designed to reduce calibration time for new users  

---

## 🧩 System Architecture
1. User wears a sensor headset  
2. Brain signals are captured and amplified  
3. Data is sent to a microcontroller (ESP32)  
4. Signals are preprocessed and classified  
5. Output is generated as text or code
