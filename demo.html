about synaplink

Project details, team, and research outcomes

project overview

SynapLink is a Brain-Computer Interface (BCI) system that converts cognitive intent from non-invasive EEG signals directly into executable code, text, and image generation prompts — without any motor input, voice, or keyboard interaction.

hardware design

  • ADS1299EEG-FE — 8-channel 24-bit EEG amplifier (0.5mW — zero heat)
  • ESP32-WROOM-32E — BLE 5.0 wireless transmission
  • Dry Ag/AgCl electrodes — no gel, no skin irritation
  • 3D-printed flexible TPU frame — 180g total weight
  • 500mAh LiPo battery — 8 hours runtime
